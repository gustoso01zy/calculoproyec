3. Flujo de uso en campo

  1. Conecta tu PC o movil al Wi-Fi del ESP32.
  2. Abre http://192.168.4.1.
  3. Inicia stream local de PC y cargalo en la web.
  4. Haz clic sobre el objetivo para registrar puntos.
  5. Revisa prediccion lineal y cuadratica.
  6. Cuando pase el horizonte, haz clic en la posicion real.
  7. Observa la tabla de errores absolutos.

4. Que calcula el sistema

  • Buffer circular de muestras: N = 3.
  • Interpolacion lineal: necesita 2 puntos.
  • Interpolacion cuadratica de Lagrange: necesita 3 puntos.
  • Error absoluto: 
    |Ex| = |x_real - x_pred|
|Ey| = |y_real - y_pred|
E = sqrt(Ex^2 + Ey^2)

5. Diagnostico rapido

No carga la pagina: confirma conexion a SSID `ESP32-Predictor`.

No sube firmware: prueba otro cable USB o baja baudios de subida.

COM ocupado: cierra Arduino IDE/monitor serie que use el puerto.

No aparece stream: revisa URL y que el stream sea HTTP/MJPEG visible.
